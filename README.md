# Projecte IoT - Control d'Aire i Ocupació en Complex Esportiu

Aquest repositori conté el codi i la documentació per al sistema de control ambiental i aforament d'un complex esportiu mitjançant una placa ESP32.

## 🚀 Característiques
- Monitorització de la qualitat de l'aire en temps real.
- Control d'aforament automàtic.
- Servidor web local amb actualització automàtica.
- Alertes visuals en cas d'emergència.

## 📁 Estructura del repositori
- `src/`: Codi font d'Arduino (.ino).
- `doc/`: Documentació tècnica i esquemes de connexió.
